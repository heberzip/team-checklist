TEAMCHECKLIST API

Bienvenido! Esta API te permite gestionar checklists y sus items.

Operaciones básicas

Listar checklists

GET /api/checklists

Crear un checklist

POST /api/checklists

Body JSON:

{
  "title": "Sprint demo",
  "description": "Tareas del sprint"
}

Obtener un checklist por ID

GET /api/checklists/{id}

Eliminar un checklist

DELETE /api/checklists/{id}

Listar items de un checklist

GET /api/checklists/{id}/items

Agregar un item

POST /api/checklists/{id}/items

Body JSON:

{
  "text": "Configurar Angular + Tailwind"
}

Actualizar un item

PATCH /api/checklists/{id}/items/{itemId}

Body JSON:

{
  "done": true,
  "lastEditedBy": "Ana"
}

Eliminar un item

DELETE /api/checklists/{id}/items/{itemId}
